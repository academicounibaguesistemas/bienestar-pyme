# Assets

Carpeta reservada para recursos estaticos del proyecto (imagenes, iconos, favicon, fuentes tipograficas, etc.).

Actualmente el prototipo no utiliza ningun archivo de este tipo: los iconos se representan con emoji Unicode directamente en el HTML/JS, y el logotipo es texto estilizado con CSS. Esta carpeta queda preparada para cuando se incorporen assets reales durante el rediseno de la interfaz.
